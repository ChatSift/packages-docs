NullableProperties

NullableProperties<T, K>: T | {}
  • Defined in packages/utils/src/index.ts:8

Creates a union type from 2 sub-types: T itself and T where all the K keys become null.

example
type Data = NullableProperties<{ x: string; y: string; z: string }, 'x' | 'y'>; // { x: string; y: string; z: string } | { x: string; y: null; z: null }
