ChatSift

Node.js libraries containing all sorts of utilities that ChatSift makes use of


Available packages
  • @chatsift/discord-utils - Niche utilities for working with Discord's API
  • @chatsift/readdir - Rewrite/fork of readdirp, a fast, stream based recursive version of fs.readdir
  • @chatsift/rest-utils - Tools for building RESTful APIs using polka
  • @chatsift/utils - General purpose functions and types that should really be language standard

Contributing

pnpm is made use of to manage the monorepo setup. It is expected that you have an up-to-date version of it.

Please ensure you run pnpm run lint, pnpm run build, and pnpm run test in the root before pushing your commits.

Please ensure that you follow the Code Of Conduct.

If all checks out, Submit a Pull Request
